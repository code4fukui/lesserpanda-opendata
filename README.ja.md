# lesserpanda-opendata

このオープンデータプロジェクトは、日本の鯖江市にある鯖江市西山動物園のレッサーパンダに関するデータセット、画像、および音声データを提供します。データは鯖江市の公式オープンデータプラットフォームから取得し、利用しやすいように処理およびクレンジングされています。

## ウェブアプリケーション

データを閲覧するためのシンプルな静的ウェブアプリケーションを利用できます。すべてのレッサーパンダの完全なリストをフィルタリング可能なテーブルで表示し、関連するすべての写真をギャラリーとして表示します。

- **デモ:** [鯖江市のレッサーパンダ](https://code4fukui.github.io/lesserpanda-opendata/)

## データとリソース

処理済みのデータは本リポジトリ内で提供されています:

- **[data/lesserpanda_sabae.csv](data/lesserpanda_sabae.csv)**: すべてのレッサーパンダの名前、家族関係、日付を含む、クレンジングおよび構造化されたリスト。
- **[data/lesserpanda_sabae_photo.csv](data/lesserpanda_sabae_photo.csv)**: 写真ファイルを対応するレッサーパンダのIDにマッピングしたマニフェスト。
- **[img/](img/)**: 処理および正規化されたレッサーパンダのJPG写真ディレクトリ。
- **[sound/](sound/)**: レッサーパンダの音声データのディレクトリ。

## データ再生成ワークフロー

このリポジトリのデータは、一連のDenoスクリプトを使用してソースから生成されています。

**前提条件:**
- [Deno](https://deno.land/) ランタイム

**手順:**

1.  **ソースデータのダウンロード**
    オープンデータポータルからオリジナルのレッサーパンダのリストをExcelファイルとしてダウンロードします。
    ```sh
    deno run -A downloadList.js
    ```

2.  **パンダリストの作成**
    ダウンロードしたExcelファイルをCSVに変換し、データをクレンジングし、日付を正規化して、各パンダに一意のIDを生成します。出力は `data/lesserpanda_sabae.csv` です。
    ```sh
    deno run -A makeList.js
    ```

3.  **画像アーカイブのスクレイピング**
    オープンデータポータルをスクレイピングし、写真が含まれる関連するすべてのZIPアーカイブを見つけてダウンロードします。
    ```sh
    deno run -A scrape.js
    ```

4.  **画像の正規化**
    ダウンロードしたZIPアーカイブから画像を抽出し、PNGをJPGに変換し、ファイル名を標準形式 (`<id>_<index>.jpg`) に変更して、写真のマニフェスト `data/lesserpanda_sabae_photo.csv` を作成します。
    ```sh
    deno run -A unzipPhoto.js
    ```
